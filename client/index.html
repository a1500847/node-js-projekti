This is a simple chat web application.
By chatting you can change the color theme of the application, pick a color using color picker.

Under 'Color Stats' you can see used colors (click a used color to use it) and the amount of messages sent using that color. The color with the most messages determines the current color theme.

Try it simultaneously with seperate windows! :)
